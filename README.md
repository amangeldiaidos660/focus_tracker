# Focus Tracker

Веб-приложение для отслеживания времени обучения в режиме фокуса.

## Стек

- Astro
- TypeScript
- Tailwind CSS
- Supabase Auth и PostgreSQL
- Google OAuth
- Vercel

## Реализовано

- авторизация через Google;
- привязка данных к аккаунту пользователя;
- создание и удаление фокус-групп;
- создание, архивирование и удаление задач;
- запуск фокус-сессии по выбранной задаче;
- обратный отсчёт с паузой и остановкой;
- настройка автоматических перерывов;
- сохранение сессий в Supabase;
- восстановление активной сессии после перезагрузки;
- статистика за день, неделю, месяц и всё время;
- история фокус-сессий по дням;
- выход из аккаунта.

## Важное ограничение модели

Фокус-группа не использует поле описания. Для неё хранится только название и системные поля базы данных.
